Docker Overview

Hamza Masood published on 0001-01-01
Why Do You Need Docker? Let’s say you need to create an application: The web server is created using nodejs, express. The database is created using mongoDB. The messaging system is created using redis. Finally, you might also need an orchestration system such as Ansible. Without using Docker, you may come across a number of issues: The compatability of the various application components with the underlying OS. For example, certain versions of the services might not be compatible with the operating system.
